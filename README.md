# 🚀 IA MasterCurator

O **IA MasterCurator** é um portal de curadoria de ferramentas de Inteligência Artificial totalmente gratuito. O objetivo é facilitar o acesso a tecnologias de ponta para estudantes, criadores de conteúdo e profissionais de tecnologia.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5 / CSS3**: Estrutura e estilização moderna com design responsivo.
* **JavaScript**: Interatividade e animações.
* **Particles.js**: Fundo dinâmico com efeito de partículas.
* **Google Fonts**: Tipografia "Inter" para melhor legibilidade.

---

## 📂 Categorias de IAs Incluídas

O portal organiza as ferramentas em quatro pilares principais:
1.  **Texto e Pesquisa**: ChatGPT, Claude, Perplexity.
2.  **Imagens e Arte**: Leonardo AI, Microsoft Designer, SeaArt.
3.  **Vídeo e Áudio**: ElevenLabs, Pika Labs, Suno AI.
4.  **Produtividade**: Gamma, Humata, Notion AI.
